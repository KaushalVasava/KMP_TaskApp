# KMP_TaskApp
It is a Kotlin Multi-Platform(KMP) Task app using SqlDelight, Kotlin, Jetpack compose.

# Features: 
- Colorful task
- Important task
- Delete Task
- Separate Completed Tasks section

# Tech stack:
- Kotlin
- Jetpack compose
- Kotlin Multi platform
- SqlDelight
- MVVM architecture
- Coroutine
- Kotlin Date Time  

# Type of App supports:
- Android
- IOS
- Desktop
